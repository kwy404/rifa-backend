Introdução

Esta documentação fornece informações sobre como utilizar a API do Sistema de Pagamento para criar e gerenciar pagamentos.

Conexão com o Banco de Dados

Verifique se o banco de dados está devidamente configurado e conectado antes de utilizar o sistema. Confira os logs do servidor para verificar o status da conexão.

const sequelize = new Sequelize(process.env.DB_NAME, process.env.DB_USER, process.env.DB_PASSWORD, { ... });

Criando Pagamentos

Para criar um pagamento Pix, envie uma requisição POST para /createPixPayment com os detalhes do pagamento.

POST /createPixPayment 
        {
          "amount": 100,
          "description": "Descrição do pagamento",
          "payerEmail": "pagador@example.com",
          "payerPhone": "1234567890"
        }

Tratando Notificações

As notificações do Mercado Pago são tratadas no endpoint /notification.

POST /notification

O status do pagamento é atualizado no banco de dados com base nos dados da notificação.

Números do Sorteio

Recupere os números do sorteio e seus status com base no email do usuário.

GET /raffleNumbers/:email
