👋 Introdução

Esta documentação fornece detalhes sobre como usar a API do Sistema de Pagamento para criar e gerenciar pagamentos, rifas e números de sorteio.

💽 Conexão com o Banco de Dados

Antes de usar o sistema, certifique-se de que o banco de dados esteja configurado corretamente e conectado. Verifique os registros do servidor para verificar o status da conexão.

const sequelize = new Sequelize(process.env.DB_NAME, process.env.DB_USER, process.env.DB_PASSWORD, { ... });

🎉 Criando uma Rifa

Para criar uma nova rifa, envie uma solicitação POST para /createRaffle com os detalhes da rifa.

POST /createRaffle 
        {
          "title": "Título da Rifa",
          "description": "Descrição da Rifa",
          "image": "URL da Imagem da Rifa"
        }

💸 Criando um Pagamento Pix

Para criar um pagamento Pix, envie uma solicitação POST para /createPixPayment com os detalhes do pagamento.

POST /createPixPayment 
        {
          "amount": 100,
          "raffleId": 1,
          "payerEmail": "pagador@example.com",
          "payerPhone": "1234567890"
        }

🔔 Tratando Notificações

As notificações do Mercado Pago são tratadas no endpoint /notification.

POST /notification

O status do pagamento é atualizado no banco de dados com base nos dados da notificação.

🎫 Números do Sorteio

Recupere os números do sorteio e seus status com base no e-mail do usuário.

GET /raffleNumbers/:email
