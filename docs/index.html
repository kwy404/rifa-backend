👋 Introdução

Esta documentação oferece informações sobre como usar a API do Sistema de Pagamento para criar e gerenciar pagamentos e rifas.

💽 Conexão com o Banco de Dados

Antes de usar o sistema, verifique se o banco de dados está corretamente configurado e conectado. Confira os registros do servidor para verificar o status da conexão.

🎉 Criando Rifas

Para criar uma rifa, envie uma solicitação POST para /createRaffle com os detalhes da rifa.

POST /createRaffle 
        {
          "title": "Título da Rifa",
          "description": "Descrição da Rifa",
          "image": "URL da Imagem da Rifa (opcional)"
        }

Exemplo de chamada de API usando JavaScript:

        const raffleDetails = {
          "title": "Minha Rifa",
          "description": "Uma rifa interessante",
          "image": "https://example.com/raffle.jpg"
        };

        fetch('/createRaffle', {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json'
          },
          body: JSON.stringify(raffleDetails)
        })
        .then(response => response.json())
        .then(data => {
          console.log(data);
        })
        .catch(error => {
          console.error('Erro na chamada de API:', error);
        });
